# 📊 Finance Analytics Dashboard – Power BI

This project is an interactive Finance Analytics Dashboard built using Power BI to analyze financial transactions, customer behavior, fees, taxes, and profitability across different segments and regions.

## 🎯 Business Problem

Organizations often struggle to monitor transaction performance, customer contribution, operational efficiency, and revenue trends in one place.

This dashboard provides a centralized analytical solution for real-time decision-making.

## 📌 Key KPIs

Total Amount

Total Transactions

Average Transaction Value

Total Fees

Total Tax

Year-over-Year (YoY) Growth

## 📈 Dashboard Insights

Retail customers contribute the highest transaction value.

Maharashtra is the top-performing state.

Transaction success rate dominates overall volume.

Monthly trends reveal seasonal fluctuations.

Transfer and Loan EMI transactions generate significant value.
